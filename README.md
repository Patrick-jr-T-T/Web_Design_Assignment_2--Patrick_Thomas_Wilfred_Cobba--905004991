This assignment requires us to come up with an idea for a Digital Public Good solution for a challenge in Sierra Leone and develop a responsive landing page for it.

# CropClock
CropClock is a concept landing page for a digital agricultural assistance platform designed to help farmers manage farming activities more effectively through timely reminders and simple farming guidance.
The platform focuses on solving challenges related to missed farming schedules, information overload, and difficulty accessing practical agricultural knowledge. CropClock provides features such as planting reminders, fertilizer schedules, weather alerts, and quick farming tips in a simple and user-friendly format.

# Features
- Smart farming reminders
- Weather awareness notifications
- Simple crop guidance
- Mobile-friendly interface
- Clean and accessible design

# Technologies Used
- HTML
- CSS
